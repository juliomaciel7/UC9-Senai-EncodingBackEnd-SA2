# Projeto Cadastro de Clientes - Senai Informática
---
:rocket: *Descrição:*

Repositório criado com o objetivo de desenvolver e compartilhar conteúdo sobre "Lógica de Programção", "Programação Orientada a Objetos" e "Codificação com Classes nativas do .NET, como StreamWrite e StremReader", para guardar dados
em arquivos.

O projeto consiste em um cadastro Pessoa Física e Jurídica, no Console, onde contém Classe Abstrata, Herança, Abrastração, Polimorfismo, Encapsulamento e métodos com validações de CPF, CNPJ, Data de nascimento e Tipo de endereço.

O C# é uma linguagem de programção orientada a objetos. Os quatro princípios básicos da programação orientada a objeto são:

+ *Abstração:*  Nesse princípio, modelamos os atributos e interações relevantes de entidades como classes, para definirmos um representação abstrata de um sistema
+ *Encapsulamento:* Esse princípio consiste em ocultar o estado interno e a funcionalidade de um objeto, e permitindo apenas o acesso por meio de um conjunto público de funções.
+ *Herança*: Consiste na capacidade de criar novas abstrações com base em abstrações existentes.
+ *Polimorfismo:* Consiste na capacidade de implementar propriedades e/ou métodos herdados de diferentes maneiras em várias abstrações.

Além disso, como Interfaces foram usadas no projeto, segue uma descrição sobre elas:

+ *Interfaces - defini o comportamento para vários tipos:* Contém definições para um grupo de funcionalidades relacionadas que uma classe concreta, ou seja, não abstrata e/ou um struc deve implementar.
A partir do C#8.0, uma interface pode definir uma implementação padrão para membros, mas não pode ser instanciada.

Classes StreamWriter e StreamReader

+  StreamWriter objeto para gravar um arquivo.
+  StreamReader objeto para ler e exibir um arquivo.
